# Comandos de Git

| Comando | Descripción |
|----------|------------|
| git init | Crear repositorio |
| git add . | Agregar archivos |
| git commit | Guardar cambios |

=== "Linux"

    ```bash
    git init
    ```

=== "Windows"

    ```powershell
    git init
    ```

!!! warning "Importante"
    Realiza commits frecuentes.
