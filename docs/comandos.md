# Comandos de Git

estos son unos de los comando mas utilizados en git.

| Comando | Descripción |
|----------|------------|
| git init | Crear repositorio |
| git add . | Agregar archivos |
| git commit | Guardar cambios |

=== "Linux"

    ```
    Usar Terminar
    git init
    ```

=== "Windows"

    ```
    Usar powershell
    git init
    ```

!!! warning "Importante"
    Realiza commits frecuentes.
